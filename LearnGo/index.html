Android Development

Our Android development course is a 54 hour course, designed to get you started with Android app development. Android platform is a world leading mobile operating system. NYC Cyber Group's Android Course is structured so students get a change to build multiple interactive apps. Each app is designed to show differetn functionalities and features of Android operating system. In this class you will build apps such as temperature converted, drawing game, camera app, as well as learn how to connect Android application to the MySQL database.

  • Session 1: Introduction to Android develpoment, Eclipe development environment, Android project structure.Download Time : 30 Munite
  • Lesson Titlename File - Lessons Programs.exeFile Size : 1.52 GB Download Time : 30 Munite
  • Session 2: Android Layouts, Views, Activities. Build a simple tempretature converter.Download Time : 30 Munite
  • Session 3: Android Splash activity, introduction to canvas and Android graphics, introduction to menu. Build a drawing game where color can be changed with the menu.Download Time : 30 Munite
  • Session 4: Introduction to hardware features, android permissions, and camera api. Build a camera app with few simple filters.Download Time : 30 Munite
  • Session 5: More hardware features. Microphone app, and GPS tracker.Download Time : 30 Munite
  • Session 6: Introduction to Android SQL Light Database. Build a API to query External MySQl database and display results on Android device.Download Time : 30 Munite
  • Session 7: Weather app using Weather API.Download Time : 30 Munite
